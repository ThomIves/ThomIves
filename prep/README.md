# /prep/ — Private interview prep section

Mobile-first, unlisted study material for active interviews. Generated by
`scripts/build_prep_site.py` in the CareerGrowth repo.

## How to add a new role section

1. Create the prep markdown source files under `CareerGrowth/applications/<RoleName>/`
2. Add a `build_<role>_section()` function to `scripts/build_prep_site.py`
3. Update `build_landing_page()` to add a card for the new section
4. Run `python3 scripts/build_prep_site.py`
