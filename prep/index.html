How this is organized

Each prep section follows the same pattern:

  • Main study doc — comprehensive reference, layered answers (lingo + conceptual + background)
  • Cram cheat sheet — the 5–10 things to absorb if you only have 10 minutes before a call
  • Things to come back to — running tracker of follow-up items, questions to verify, things you said you'd send

The PG&E MIDA section also includes a web version of the panel presentation — one HTML page per slide, with the speaker notes accessible from any slide. Useful for rehearsing on the phone or tablet.

Tips for studying on a phone

  • Tap the moon icon (top right) to toggle dark mode for night reading
  • On long pages, the floating table of contents (collapsible) gets you to a section fast
  • Browser's "Find on page" (long-press the address bar in Safari, Chrome menu in Android) works as a search
  • "Save as PDF" or "Print" works from any mobile browser for offline access

For future role prep

To add a new role section: create a sub-directory under prep/, drop in the source markdown files, and re-run the build script. The landing page can be updated to add the new section card. The structure is designed to be extensible.
