10-Minute PG&E MIDA Cram Cheat Sheet

The five load-bearing things to absorb if you only have 10 minutes before the next conversation.

1. What MIDA actually is — say this verbatim if asked

"MIDA is a Multi-modal Inspection Data Annotation platform built within Palantir Foundry. It uses Vision Language Models — currently Gemini 3.1, parameter-tuned for zero-shot — to identify electric assets in drone, helicopter, and satellite imagery: poles, transformers, splices, circuits, attachments. The platform is human-in-the-loop by regulatory mandate; AI surfaces candidates, the human inspector decides. The Golden Pipeline is the template-based architecture that lets new asset types be added through configuration rather than code."

2. The split — VLMs vs Waldo's CNNs

VLMs (Gemini 3.1) do the first-pass general detection. Waldo is PG&E's internal team building traditional trained CV models (CNNs) for the deeper condition-assessment work where VLMs hit their accuracy ceiling. The role would be backfilling a Waldo team member, working on production CV + MLOps to get models from concept to production.

3. The four metrics for the technical screen with Erin (and probably Charles)

  • Precision = TP / (TP + FP) — "when the model says yes, is it right?"
  • Recall = TP / (TP + FN) — "of the real positives, how many did the model catch?"
  • F1 = harmonic mean of precision and recall — "is the model both precise AND complete?"
  • mAP = mean Average Precision across IoU thresholds — "detection + localization combined"

The EPIC 4.22 deployment bar is F1 > 0.8. BPE testing found only 9% of out-of-box models hit that bar.

4. The validation set trap (the Adam-flagged one)

Don't randomly split video frames — neighboring frames are nearly identical, model memorizes. Split by time first, then by physical asset and camera. For rare classes, synthetic / AI-generated training data is acceptable; test set stays real.

5. Your strongest bridge to MIDA's work

The ON Semi Streak Test Auto-Detector: production CV defect detection on a manufacturing line under 3-week release cycles. Iterative robust regression — the methodology lineage from the M.S. robot calibration research. "Auto-detector replacing the human's slow tuning loop is the same problem shape MIDA is solving today, applied to electrical asset imagery instead of imaging die."

Plus: Microsoft Fabric AI Functions classifier with gpt-4.1-mini at million-row scale, evaluation scaffolding, structured outputs, per-row cost/latency management — same production discipline applied to the multimodal foundation model family that VLMs sit in.

Show depth. Don't pile credentials — anchor every technical answer in a specific past project.
