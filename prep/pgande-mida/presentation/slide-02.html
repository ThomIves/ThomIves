PITAD

Production Image Test Algorithm Development

at ON Semi

  • What we did: tested die on the wafer, before packaging
  • Why it mattered: catch bad die before they ship to customers
  • Production discipline:
    • Feedback from production drove test refinements
  • Updated test code ran through an established + growing test system
    • Releases went out per the established cadence
  • Per-product test groups — each product had its own configured test suite

ON Semi's PITAD team — Production Image Test Algorithm Development — built and maintained the computer-vision tests that ran against die imaged while still on the wafer, before packaging. The point was simple: catch bad die before we sold them. We had a continuous improvement loop — feedback from production drove test refinements, which ran through an established test system, which released on cadence. Every product got its own group of tests.
