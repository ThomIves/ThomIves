What I Built — Iterative Robust Regression for Streak Defect Detection

  • Parametric illumination model — 5th-degree bivariate polynomial surface fit to the imaging system's background dome
  • Residual map = actual pixels − model — defects live in the residuals
  • Iterative outlier suppression with refit until convergence:
    • Identify pixels outside N standard deviations from the fit
  • Suppress those pixels; refit the polynomial surface on the remaining
    • Find the new outliers; repeat until none remain
  • Methodology lineage: iterative robust least-squares drawn from my M.S. robot-calibration research at Texas A&M
  • Spatial clustering / connected-component analysis on the residual anomaly map (size + density thresholds from related cluster-defect tests)
  • Engineering constraint: Python 2.5 + numpy only (no SciPy) — numerical routines written from scratch

LOAD-BEARING SENTENCE TO MEMORIZE THE SHAPE OF:

"I built a defect detector based on a parametric illumination model — a 5th-degree bivariate polynomial surface fit to the imaging system's background dome. Defects live in the residual map between the model and the actual pixels. I iterated robust regression — fit, suppress outliers, refit — until convergence, then ran spatial clustering on the residual anomalies with size and density thresholds borrowed from related cluster-defect tests."

Then expand naturally:

"A practical note on why this approach was possible — the iterative robust-least-squares machinery came from my graduate work on robot calibration at Texas A&M. We used iteratively refined least-squares fits with outlier suppression to calibrate production robots against sensor measurements. The same statistical methodology applied to the Streak Test image surface twenty years later. Different physical system, same machinery."

"On the iteration itself — fitting a polynomial surface once is easy. The engineering insight is that one fit is not enough when defects bias the fit. Iterating fit-suppress-refit until convergence lets the model find the true background even when defects are present in the image. That iteration is what makes the detector robust."

"One engineering-constraint note — the test system was stuck on Python 2.5 with numpy, no SciPy. So I wrote the linear-algebra and statistical routines I needed from scratch in numpy. It became a personal predictive-modeler war story. The discipline came with the constraint."
