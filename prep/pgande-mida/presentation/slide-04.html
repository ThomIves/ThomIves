What I Built — Iterative Robust Regression + Pseudo-Gravitational Clustering

– Delta-E-informed acceptable-range test (color-perception standard)

– Suppress out-of-range pixels; refit; repeat until all remaining pixels fall in range

  • Parametric illumination model — 5th-degree bivariate polynomial surface fit to the imaging system's background dome
  • Pixels contrast-enhanced first; residual map = actual pixels − model — defects live in the residuals
  • Iterative robust regression with outlier suppression until convergence:
  • Methodology lineage: iterative robust least-squares from my M.S. robot-calibration research at Texas A&M
  • Clustering the suspect pixels: KNN was blocked — automating k-selection isn't tractable
  • Invented Pseudo Gravitational Attraction (PGA) with per-pixel Pseudo Mass (PM based on model deviation)
  • Groups exceeding each product's Smallest Objectionable Defect (SOD) threshold — from Outgoing Defect Specs — fail the die
  • Engineering constraint: Python 2.5 + numpy only (no SciPy) — numerical routines written from scratch

LOAD-BEARING SENTENCE TO MEMORIZE THE SHAPE OF:

"I built a defect detector based on a parametric illumination model — a 5th-degree bivariate polynomial surface fit to the imaging system's background dome. Pixels were contrast-enhanced first; defects live in the residual map between the model and the actual pixels. I iterated robust regression — fit, suppress outliers, refit — until convergence, using a Delta-E-informed test (the color-perception standard) to define the acceptable intensity range at each iteration. What was left out of range were the suspect pixels."

Then continue with the invention:

"Clustering the suspects was the hard part. KNN is the obvious tool but automating the choice of k isn't tractable — how would you pick k automatically for every product? So I invented a method I called Pseudo Gravitational Attraction. Each suspect pixel gets a Pseudo Mass proportional to how far its intensity deviates from the model. A suspect that has enough gravitational attraction toward an existing cluster joins that cluster. When a cluster's total mass and extent exceed the product's Smallest Objectionable Defect threshold — that's a per-product number that already existed in our Outgoing Defect Specifications — the die fails."

The intellectual point (say it if the room is engaged):

"The lesson here is worth naming. When a standard method blocks you because a parameter can't be automated, invent something in-domain that can. KNN is elegant but k has to be chosen. PGA doesn't choose k — the groups form themselves from the product's SOD, which the business already had a number for."

Methodology-lineage note:

"The iterative robust-least-squares machinery came from my graduate work on robot calibration at Texas A&M. We used iteratively refined fits with outlier suppression to calibrate production robots against sensor measurements. Twenty years later, same statistical machinery on the Streak Test image surface. Different physical system, same discipline."

Engineering constraint (short):

"The test system was stuck on Python 2.5 with numpy, no SciPy. So I wrote the linear-algebra and statistical routines from scratch in numpy. The discipline came with the constraint."
