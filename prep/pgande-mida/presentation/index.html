Quick reference — load-bearing sentence (Slide 4)

Memorize the SHAPE of this

"I built a defect detector based on a parametric illumination model — a 5th-degree bivariate polynomial surface fit to the imaging system's background dome. Pixels were contrast-enhanced first; defects live in the residual map between the model and the actual pixels. I iterated robust regression until convergence using a Delta-E-informed test to define the acceptable intensity range. The suspect pixels — those still outside the range at convergence — were clustered using a novel method I called Pseudo Gravitational Attraction, with per-pixel Pseudo Mass based on model deviation. Groups exceeding each product's Smallest Objectionable Defect threshold failed the die."
